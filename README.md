# 東大ドラえもんF同好会のウェブサイト

アクティビティを追加したい場合は、
1. assets/data/activities.jsonにアクティビティの情報を追加する
   - featuredをtrueにするとトップページに表示される
2. activities/にアクティビティのページを作成する
   - テンプレートは、activities/report_template.htmlを参考にしてください
   - ページのファイル名は、260331_Kyodai.htmlのようにする
3. assets/images/activities/にアクティビティの画像を追加する
   - 画像のファイル名も、260331_Kyodai.jpegのようにする
4. 必須ではないが、sitemap.xmlを更新する
